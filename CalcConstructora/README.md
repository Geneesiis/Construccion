CalcConstructora
=================

Pequeña app móvil con calculadora para construcción.

Qué hay en este repositorio
- `App.js`, `index.js`, `package.json`: punto de entrada y configuración.
- `src/`: código fuente (componentes, base de datos, estilos).
- `assets/`: imágenes y recursos.
- `AGENTS.md`: nota sobre la versión de Expo a consultar.

Cómo ejecutar (desarrollo)
1. Instalar dependencias:

```bash
npm install
```

2. Iniciar Metro/Expo:

```bash
npx expo start
```

Nota importante
- Antes de publicar o cambiar configuraciones de Expo, revisa la documentación versionada indicada en `AGENTS.md` (por ejemplo, https://docs.expo.dev/versions/v54.0.0/).
